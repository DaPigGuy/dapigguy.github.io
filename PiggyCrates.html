PiggyCrates

PiggyCrates's documentation page includes the configuration documentation.

  • Information

    PiggyCrates is a simple and customizable crates plugin with roulettes, supporting both custom and vanilla enchantments.

    As of October 19, 2019, PiggyCrates has been completely rewritten from scratch. Click the box below to learn more.

    Version 2.0.0 - changes

    Version 2.0.0 is a complete rewrite of the plugin and introduces tons of bug fixes, optimizations, and quality of life changes.

    • Now using Commando virion for command handling.
    • Crates are now saved as tiles.
    • Floating text particles have been added.
    • Crate roulettes have been added.
    • Crate items now support NBT.
    • Crate items now support custom name/lore.
  • Configuration

    Default Configuration

    Explanation

    Key Index Explanation
    keys id Specifies the item that should be used to act as a "key" for the crate.

    Accepted values: item id
    meta Specifies the meta value for the key item.

    Accepted values: item meta
    name Specifies the name of the key item.

    Accepted values: item name
    lore Specifies the lore (the text under the name) of the key item.
    You can add multiple lines in lore by adding \n.
    Example: lore: "Line #1\nLine #2\nLine #3"

    Accepted values: item lore
    crates mode Whether to instantly give items or open the roulette menu.
    If roulette is selected, you can define duration and speed of roulette in the following section.

    Accepted values: instant, roulette
    roulette.duration Determines the length of the roulette in ticks (20 ticks = 1 second).

    Accepted values: roulette duration
    roulette.speed Determines the speed at which the roulette items should shift left.

    Accepted values: roulette speed
    Crates Configuration

    Default Configuration

    Explanation

    Key Index Data Explanation
    crates.crate_type drops id Specifies the id of the drop item.

    Accepted values: item id
    meta Specifies the meta of the drop item.

    Accepted values: item meta
    amount Specifies the amount of items that should be given.

    Accepted values: number of items
    chance Specifies the chance of recieving the drop item.
    The accumulative chance values of all drops should equal to 100.

    Accepted values: drop chance
    name Specifies the name of the drop item. (optional)

    Accepted values: item name
    lore Specifies the lore (the text under the name) of the drop item. (optional)

    Accepted values: item lore
    nbt Specifies NBT tags on the drop item. (optional)
    Any JSON-ified strings will work for NBT tags. Refer here for possible vanilla NBT tags.

    Accepted values: json
    enchants Specifies vanilla or custom enchants to apply onto the drop item. (optional)
    PiggyCustomEnchants support is added by default.

    Accepted values: enchants => name, level
    amount Specifies the amount of times that the crate should roll with only one key.

    Accepted values: number of rolls
    drops Specifies the content of the floating text above the crate.

    Accepted values: text
