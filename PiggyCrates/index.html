Introduction

PiggyCrates is a simple, customizable, crates plugin. This plugin allows for infinite amounts of crate types. We have support for Custom Enchants, as well as normal enchants. You can customize the drops, names of drops, lore of drops, chances of drops, and much more. Below is a thorough explanation of the formatting of crate types, as well as examples.

Configuration Guide

    Customization is one of the reasons PiggyCrates is a great crates plugin. Below is a through explanation of the formatting of crate types, and what you can do with this crate plugin.

    Breakdown of crates.yml

      
    crates:
      example: (crate type - name of the crate)
        drops: (items that can be received from the crate)
          - name: "Example Item" (name of item)
            id: (id of drop)
            meta: 0 (meta/dmg value of item)
            amount: 10 (amount of items that are received from this drop)
            chance: 100 (chance of item to drop)
            enchantments: (enchantments on item)
              enchantment1: (an enchantment)
                level: 1 (level of enchantment)
              enchantment2: (another enchantment)
                level: 1 (level of enchantment)
            lore: Example Lore (adds a description to the item; you can use \n for a new line)
            command: tell %player% You opened the Common Crate! (command run when item drops; you can use %PLAYER% if you want to add the name of the player)
        amount: 1 (amount of drops that can be received from opening crate)
        block: "173:0" (id:meta; the block under chest to determine crate type)

    Breakdown of config.yml

      
    key: "131:0" (id:meta; key item)
    key-lore: "Use this key to get rewards." (description of key)
    allow-crate-changes: false (allow changes to crates; creation and destruction of crates without the permission piggycrates.crates)

Examples

    Below are examples of different crate types, including a crate with no enchants, a crate with vanilla enchants, and a crate with custom enchants.

    Example with No Enchants

      
    crates:
      Voting:
        drops:
          - name: "Unlucky Dirt"
            id: 3
            meta: 0
            amount: 1
            chance: 50
          - name: "Lucky Diamond"
            id: 264
            meta: 0
            amount: 1
            chance: 10
          - name: "Rusty Sword"
            id: 267
            meta: 200
            amount: 1
            chance: 40
      amount: 1
      block: "42:0"

    Example with Vanilla Enchants

      
    crates:
      Uncommon:
        drops:
          - name: "Knockback Stick"
            id: 280
            meta: 0
            amount: 1
            chance: 50
            enchantments:
              Knockback:
                level: 2
          - name: "Lucky Diamond"
            id: 264
            meta: 0
            amount: 1
            chance: 25
            enchantments:
              Efficiency:
                level: 5
          - name: "Shiny Sword"
            id: 267
            meta: 0
            amount: 1
            chance: 25
            enchantments:
              Sharpness:
                level: 1
              Unbreaking:
                level: 1
      amount: 1
      block: "42:0"

    Example with Custom Enchants

      
    crates:
      Rare:
        drops:
          - name: "Piggy Slayer"
            id: 261
            meta: 0
            amount: 1
            chance: 40
            enchantments:
              Porkified:
                level: 3
              Paralyze:
                level: 1
          - name: "Lucky Diamond"
            id: 264
            meta: 0
            amount: 1
            chance: 20
            enchantments:
              Soulbound:
                level: 1
          - name: "Deathbringer"
            id: 268
            meta: 50
            amount: 1
            chance: 40
            enchantments:
              Deathbringer:
                level: 2
              Disarmor:
                level: 1
      amount: 2
      block: "133:0"
