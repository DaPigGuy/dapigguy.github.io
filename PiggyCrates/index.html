    Configuration Guide:

    Default Configuration

    
        crates:
          example: #(crate type - name of the crate)
            drops: #(items that can be received from the crate)
              1: #(id of drop)
                name: "Example Item" #(name of item)
                meta: 0 #(meta/dmg value of item)
                amount: 10 #(amount of items that are received from this drop)
                enchantments:
                  enchantment1:
                    level: 1
                  enchantment2:
                    level: 1
            amount: 1 #(amount of drops that can be received from opening crate)
            block: "173:0" #(id:meta; the block under chest to determine crate type)

    You are not limited to only two enchantments, this plugin allows for unlimited enchants as long as the enchantments are registered into pmmp.
    In other words, if you are using other enchantment plugins other than PiggyCustomEnchants or PMMP, those enchantments will also work.

    Example Configuration with/without Custom Enchants:

    With custom enchants, simply refer to the second drop item.
    Without custom enchants, simply refer to the first drop item.

    
        crates:
          example:
            drops:
              3:
                name: "Unlucky Item"
                meta: 0
                amount: 32
              268:
                name: "Sword of Death"
                meta: 0
                amount: 1
                enchantments: 
                  Deathbringer:
                    level: 1
                  Disarmor:
                    level: 1
            amount: 1
            block: "133:0"
