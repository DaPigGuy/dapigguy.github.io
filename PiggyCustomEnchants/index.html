  • The API Documentation of PiggyCE.

API Documentation - Usage

Usage of $ce->addEnchantment():

            
            $ce->addEnchantment($item, $enchants, $levels, $check);
            // $ce is the plugin.
            // $item is the item.
            // $enchants can be an array, or a single value. Enchantment names, ids, or classes can be used.
            // $levels can be an array, or a single value, but has to be an integer.
            // $checks should be a boolean - true for checks, false for none. default = true.

Usage of $ce->removeEnchantment():

            
            $ce->removeEnchantment($item, $enchant, $level);
            // $ce is the plugin.
            // $item is the item.
            // $enchant is the name of the enchant.
            // $level is not needed unless you want to remove an enchant with a certain level. (optional)
            // $ce->removeEnchantment() should be instanceof EnchantmentInstance or CustomEnchants.

Usage of $item->getEnchantment():

            
            $item->getEnchantment($id);
            // $item is the item.
            // $id is the id of the enchant.

Usage of $ce->canBeEnchanted():

            
            $ce->canBeEnchanted($item, $enchant, $level);
            // $ce is the plugin.
            // $item is the item.
            // $enchant is the name of the enchant.
            // $level of enchant.
            // $ce->removeEnchantment() should be instanceof EnchantmentInstance or CustomEnchants.

Usage of $ce->registerEnchantment():

            
            $ce->registerEnchantment($id, $name, $type, $trigger, $rarity, $maxlevel, $description = "");
            // $ce is the plugin.
            // $id is the id of custom enchant to be registered.
            // $name is the name of custom enchant.
            // $type is the item or item type which it can be applied on.
            // $trigger is what triggers the custom enchant to activate.
            // $rarity is the level of rarity of custom enchant.
            // $maxlevel is the max level of custom enchant.
            // $description is the description of custom enchant. (optional)

Usage of $ce->unregisterEnchantment():

            
            $ce->unregisterEnchantment($id);
            // $ce is the plugin.
            // $id is the id of custom enchant to be unregistered.

API Documentation - Examples

Below are examples of various methods the API can be used for. Reminder: These are EXAMPLES, so not everything should be copied as-is.

To add custom enchant(s) to the current item player is holding:

            
            $item = $player->getInventory->getItemInHand(); //$player is the player
            $ce = $this->getServer()->getPluginManager()->getPlugin("PiggyCustomEnchants");
            $ce->addEnchantment($item, ["Porkified", "Grappling"], [3, 1], $check);
            $player->getInventory()->setItemInHand($item);

To add custom enchant(s) to an item in a specific slot (slot 0):

            
            $item = $player->getInventory->getItem(0); //$player is the player
            $ce = $this->getServer()->getPluginManager()->getPlugin("PiggyCustomEnchants");
            $ce->addEnchantment($item, ["Porkified", "Grappling"], [3, 1], $check);
            $player->getInventory()->setItem(0, $item);

To add custom enchant(s) to a preexisting item with custom enchants:

            
            $item = Item::get(Item::BOW);
            $ce = $this->getServer()->getPluginManager()->getPlugin("PiggyCustomEnchants");
            $ce->addEnchantment($item, ["Porkified", "Grappling"], [3, 1], $check);
            $player->getInventory()->addItem($item); //$player is the player

To remove custom enchant(s) to the current item player is holding:

            
            $item = $player->getInventory()->getItemInHand(); //$player is the player
            $ce = $this->getServer()->getPluginManager()->getPlugin("PiggyCustomEnchants");
            $ce->removeEnchantment($item, CustomEnchantsIds::PORKIFIED, $level);
            $player->getInventory()->setItemInHand($item);

To remove custom enchant(s) to an item in a specific slot (slot 0):

            
            $item = $player->getInventory()->getItem(0); //$player is the player
            $ce = $this->getServer()->getPluginManager()->getPlugin("PiggyCustomEnchants");
            $ce->removeEnchantment($item, CustomEnchantsIds::PORKIFIED, $level);
            $player->getInventory()->setItem(0, $item);

To check if an item has an enchant:

            
            $enchantment = $item->getEnchantment(CustomEnchantsIds::PORKIFIED);
            if($enchantment !== null){
                // item has enchantment
            }

To check if item can be enchanted:

            
            $ce = $this->getServer()->getPluginManager()->getPlugin("PiggyCustomEnchants");
            if($ce->canBeEnchanted($item, CustomEnchantsIds::PORKIFIED, 3)){
                // can be enchanted
            }

To register your own enchant:

            
            $ce = $this->getServer()->getPluginManager()->getPlugin("PiggyCustomEnchants");
            $ce->registerEnchantment(302, "Fireworks", "Bow", "Projectile_Hit", "Uncommon", 1, "Launches fireworks on hit");

To unregister an enchant:

            
            $ce = $this->getServer()->getPluginManager()->getPlugin("PiggyCustomEnchants");
            $ce->unregisterEnchantment(302);
