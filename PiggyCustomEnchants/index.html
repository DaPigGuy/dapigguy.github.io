    API Documentation:

    To add a custom enchantment (Porkified III and Grappling I) to the item player is holding:

    
        $item = $player->getInventory->getItemInHand(); //$player is the player
        $ce = $this->getServer()->getPluginManager()->getPlugin("PiggyCustomEnchants");
        $ce->addEnchantment($item, ["Porkified", "Grappling"], [3, 1], $check); //Enchants and level can be arrays or one specific value. Check arguments is not needed unless you would like to override checks (set to true). Enchant names, ids, or classes can be used.
        $player->getInventory()->setItemInHand($item);

    To add a custom enchant (Porkified III and Grappling I) to an item in a certain slot (slot 0):

    
        $item = $player->getInventory->getItem(0); //$player is the player
        $ce = $this->getServer()->getPluginManager()->getPlugin("PiggyCustomEnchants");
        $ce->addEnchantment($item, ["Porkified", "Grappling"], [3, 1], $check); //Enchants and level can be arrays or one specific value. Check arguments is not needed unless you would like to override checks (set to true). Enchant names, ids, or classes can be used.
        $player->getInventory()->setItem(0, $item);

    To add an item with custom enchants (Porkified III and Grappling I):

    
        $item = Item::get(Item::BOW);
        $ce = $this->getServer()->getPluginManager()->getPlugin("PiggyCustomEnchants");
        $ce->addEnchantment($item, ["Porkified", "Grappling"], [3, 1], $check); //Enchants and level can be arrays or one specific value. Check arguments is not needed unless you would like to override checks (set to true). Enchant names, ids, or classes can be used.
        $player->getInventory()->addItem($item); //$player is the player

    To remove a custom enchantment (Porkified) from the item player is holding:

    
        $item = $player->getInventory()->getItemInHand(); //$player is the player
        $ce = $this->getServer()->getPluginManager()->getPlugin("PiggyCustomEnchants");
        $ce->removeEnchantment($item, CustomEnchantsIds::PORKIFIED, $level); //Level argument is not needed unless you would like to remove an enchant with a certain level
        $player->getInventory()->setItemInHand($item);

    To remove a custom enchant (Porkified) for an item in a certain slot (slot 0):

    
        $item = $player->getInventory()->getItem(0); //$player is the player
        $ce = $this->getServer()->getPluginManager()->getPlugin("PiggyCustomEnchants");
        $ce->removeEnchantment($item, CustomEnchantsIds::PORKIFIED, $level); //Level argument is not needed unless you would like to remove an enchant with a certain level
        $player->getInventory()->setItem(0, $item);

    To check if a item has an enchant:

    
        $enchantment = $item->getEnchantment(CustomEnchantsIds::PORKIFIED); //$item is the item
        if($enchantment !== null){
            //Item has enchantment
        }

    To check if you can enchant the item:

    
        $ce = $this->getServer()->getPluginManager()->getPlugin("PiggyCustomEnchants");
        if($ce->canBeEnchanted($item, CustomEnchantsIds::PORKIFIED, 3)){
            //Can be enchanted
        }

    To register your own enchant:

    
        $ce = $this->getServer()->getPluginManager()->getPlugin("PiggyCustomEnchants");
        $ce->registerEnchantment(302, "Fireworks", "Bow", "Projectile_Hit", "Uncommon", 1, "Launches fireworks on hit");

    To unregister an enchant:

    
        $ce = $this->getServer()->getPluginManager()->getPlugin("PiggyCustomEnchants");
        $ce->unregisterEnchantment(302);
