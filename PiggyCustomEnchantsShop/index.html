Plugin Installation

    Plugin Installation (Signs)

      Steps
      1. Ensure you have basic knowledge of opening directories, editing files, reading steps, and understand the fundamentals of typing with a keyboard.
      2. Have a functional PMMP server instance, running the latest version of PMMP. If you do not have the latest version of PMMP, download it at the PMMP Jenkins site.
      3. Download the plugins listed below and drop them into your plugins folder.
        • PiggyCustomEnchants
        • PiggyCustomEnchantsShop
      4. Start your PMMP server.

    Plugin Installation (UI)

      Steps
      1. Ensure you have basic knowledge of opening directories, editing files, reading steps, and understand the fundamentals of typing with a keyboard.
      2. Have a functional PMMP server instance, running the latest version of PMMP. If you do not have the latest version of PMMP, download it at the PMMP Jenkins site.
      3. Download the plugins listed below and drop them into your plugins folder.
        • PiggyCustomEnchants
        • PiggyCustomEnchantsShop
        • FormAPI
      4. Start your PMMP server.
      5. In the config.yml, set shop-type: sign to shop-type: ui. It should look something like this: 
        
    double-tap: true
    shop-type: ui
      6. Restart your PMMP server.

Shop Setup

    Shop Setup (Signs)

      Steps
      1. Make sure you have shop-type set to sign in the config.yml.
      2. Place a sign.
      3. Put CE on the first line.
      4. Put the enchantment on the second line.
      5. Put the enchantment level on the third line.
      6. Put the price on the last line.

    Shop Setup (UI)

      Steps
      1. Make sure you have shop-type set to ui in the config.yml.
      2. Use the command /ceshop add <enchant> <level> <price>
