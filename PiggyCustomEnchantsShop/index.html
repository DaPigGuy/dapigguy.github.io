PiggyCustomEnchantsShop

Plugin Installation

On this page, we will walk you through the process of installing the PiggyCustomEnchantsShop plugin with FormsAPI by jojoe77777, an optional addition to the PiggyCustomEnchantsShop plugin.

Step 0: Ensure you have basic knowledge of opening directories, editing files, reading the steps, and understand the fundamentals of typing with a keyboard.

Step 1: Have a functional pmmp server instance, running the latest version of pmmp. If you do not have the latest version of pmmp, download it at their Jenkins site.

Step 2: Download the plugins listed below, then drop them into your /plugins folder.

  • PiggyCustomEnchants
  • PiggyCustomEnchantsShop
  • FormAPI

Step 3: Start your pmmp server.

Step 4: In the file /plugins/PiggyCustomEnchantsShop/config.yml, set shop-type: sign to shop-type: ui

Step 5: Start your pmmp server, and PiggyCustomEnchantsShop with FormsAPI has been setup!
