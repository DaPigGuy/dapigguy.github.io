      createGame()
        Description
          createGame() creates a game instance.
        Usage
          
    $gameManager->createGame($name, $floors, $spawn, $respawn, $lobby, $hub, $world, $minPlayers, $maxPlayers, $countdownTime, $snowballsEnabled, $effects, $winnerPrizes, $winnerCommands);
        Parameters
          Name
          • The name of the arena
          Floors
          • The floors of the arena. Must be an array consisting of arrays consisting of 2 arrays with the keys start and end, each with 3 values for the x, y, & z axis of the positions.
          Spawn
          • The spawn of the game. Must be an array consisting of 3 values for the x, y, & z axis of the positions.
          Respawn
          • The respawn of the game. Must be an array consisting of 3 values for the x, y, & z axis of the positions.
          Lobby
          • The lobby (waiting room) of the game. Must be an array consisting of 4 values for the x, y, & z axis of the positions, as well as the world name.
          Hub
          • The hub (main spawn) of the game. Must be an array consisting of 4 values for the x, y, & z axis of the positions, as well as the world name.
          World
          • The world name of the arena.
          Minimum Players
          • The minimum amount of players needed for the game to start.
          Maximum Players
          • The maximum amount of players allowed in a game.
          Countdown Time
          • The amount of time for the countdown.
          Snowballs Enabled
          • Whether or not to have snowballs enabled.
          Effects
          • The effects granted at the beginning of the game. Must be an array. Values must be strings, formatted as id:level:duration
          Winner Prizes
          • The items granted to the winner of the game. Must be an array. Values must be strings, formatted as id:damage:amount
          Winner Commands
          • The commands used at the end of the game. Must be an array. Values must be strings.
        Example
          
    $floors = [["start" => ["startX" => -5, "startY" => 1, "startZ" => -5],
                "end" => ["endX" => 5, "endY" => 1, "endZ" => 5]];
    $spawn = ["spawnX" => 1, "spawnY" => 1, "spawnZ" => 1];
    $respawn = ["respawnX" => 1, "respawnY" => 1, "respawnZ" => 1];
    $lobby = ["lobbyX" => 11, "lobbyY" => 10, "lobbyZ" => 11, "lobbyWorld" => "world"];
    $hub = ["hubX" => 50, "hubY" => 10, "hubZ" => 50, "hubWorld" => "world"];
    $spleef = $this->getServer()->getPluginManager()->getPlugin("PiggySpleef");
    $spleef->createGame("ArenaOne", $floors, $spawn, $respawn, $lobby, $hub, "world", 5, 15, false, ["1:1:30"], ["264:0:2"], ["say {PLAYER} has won a game in ArenaOne!"])
