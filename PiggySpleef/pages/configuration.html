PiggySpleef

    Configuration Guide

      
    arenas:
      arenaName:
        floors:
          - start:
              startX: 0
              startY: 0
              startZ: 0
            end:
              endX: 0
              endY: 0
              endZ: 0
        minPlayers: 3
        maxPlayers: 15
        countdownTime: 30
        snowballsEnabled: false
        effects:
          - "1:1:60"
          - "3:1:60"
        winnerPrizes:
          - "264:0:2"
          - "388:0:1"
        winnerCommands:
          - "say {PLAYER} has won a game in arenaName!"
        spawnPos:
          spawnX: 0
          spawnY: 0
          spawnZ: 0
        respawnPos:
          respawnX: 0
          respawnY: 0
          respawnZ: 0
        lobbyPos:
          lobbyX: 0
          lobbyY: 0
          lobbyZ: 0
          lobbyWorld: world
        hubPos:
          hubX: 0
          hubY: 0
          hubZ: 0
          hubWorld: world
        world: world

    Breakdown of Configuration

      arenas
      • this section specifies what arenas are available. (array)
      arenaName
      • name of the arena. to create another arena, simply copy and paste the configuration following "arenaName" after. (string)
      floors
      • defines the start and end floors of the arena. (array)
        • start
          • starting floor of the arena. you are required to fill three parameters. (array)
            • startX - starting x-axis value (int)
            • startY - starting y-axis value (int)
            • startZ - starting z-axis value (int)
        • end
          • the ending floor of the arena. you are required to fill three parameters. (array)
            • endX - ending x-axis value (int)
            • endY - ending y-axis value (int)
            • endZ - ending z-axis value (int)
      minPlayers
      • Minimum Players needed to start a Spleef Game. (int)
      maxPlayers
      • Maximum Players needed to start a Spleef Game. (int)
      countdownTime
      • Amount of Time before Spleef Game starts. (int)
      snowballsEnabled
      • Whether or not to enable Snowballs in the arena. (true/false)
      effects
      • List of Effects to give players upon joining a Spleef Game. (array)
      winnerPrizes
      • List of Prizes to award the player who wins a Spleef Game. (array)
      winnerCommands
      • List of Commands to execute when a player has won a Spleef Game. (array)
      spawnPos
      • the spawn position of all players for the arena.
        • spawnX - spawn x-axis value (int)
        • spawnY - spawn y-axis value (int)
        • spawnZ - spawn z-axis value (int)
      respawnPos
      • the respawn position of all players for the arena.
        • respawnX - respawn x-axis value (int)
        • respawnY - respawn y-axis value (int)
        • respawnZ - respawn z-axis value (int)
      lobbyPos
      • the spawn position of all players for the arena.
        • lobbyX - lobby x-axis value (int)
        • lobbyY - lobby y-axis value (int)
        • lobbyZ - lobby z-axis value (int)
        • lobbyWorld - world lobby is in. this is the waiting area before entering arena. it can be the same as hub world, if wanted. (string)
      hubPos
      • the spawn position of all players for the arena.
        • hubX - hub x-axis value (int)
        • hubY - hub y-axis value (int)
        • hubZ - hub z-axis value (int)
        • hubWorld - world hub is in
      world
      • world which the arena is located in.
