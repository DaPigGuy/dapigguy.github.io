Welcome to PiggySpleef's Documentation! You can find the setup process for the plugin on this page!

Plugin Installation

    Steps

    1. Ensure you have basic knowledge of opening directories, editing files, reading steps, and understand the fundamentals of typing with a keyboard.
    2. Have a functional PMMP server instance, running the latest version of PMMP. If you do not have the latest version of PMMP, download it at the PMMP Jenkins site.
    3. Download the plugin and drop it into your plugins folder.
    4. Start your PMMP server.
    5. In the command console, type in "pl" to ensure that the plugin is green (active) and not red (inactive).
    • Although the plugin has loaded, additional setup is required to:
      • create a spleef arena
      • configure your spleef arena settings
      • and etc.
      Refer to PiggySpleef's Configuration Guide for a breakdown of the configuration!
