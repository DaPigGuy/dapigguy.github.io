    Plugin Installation

    On this page, we will walk you through the process of installing the PiggyCustomEnchantsShop plugin, as it requires FormsAPI by jojoe77777 in order to work with the UI.

    Step 0: Ensure you have basic knowledge of opening directories, editing files, reading the steps, and understand the fundamentals of typing with a keyboard.

    Step 1: Have a functional pmmp server instance, running the latest version of pmmp. If you do not have the latest version of pmmp, download it at their Jenkins site.

    Step 2: Download DevTools and drop DevTools_v?.phar into your /plugins folder.

    Step 3: Download the plugins listed below, then drop them into your /plugins folder.

    • PiggyCustomEnchants
    • PiggyCustomEnchantsShop
    • FormAPI

    Step 4: Start your pmmp server.

    Step 5: On the console window, execute the command makeplugin FormAPI, then stop the server.

    Step 6: Delete FormAPI-master folder, extract the FormAPI.phar file from /plugins/DevTools and drop it into the /plugins folder

    Step 7: In the file /plugins/PiggyCustomEnchantsShop/config.yml, set shop-type: sign to shop-type: ui

    Step 8: Start your pmmp server, and PiggyCustomEnchantsShop with FormsAPI has been setup!
